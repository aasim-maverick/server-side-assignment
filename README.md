# root-repo

Server-side assignment archive with clean, modular solutions across HTML fundamentals and JavaScript problem solving.

## What this repo contains

- `assignment-1`: foundational web development exercises plus two multi-page mini websites.
- `assignment-2`: 25 JavaScript solutions covering algorithms, data structures, object handling, and browser basics.

## Why this stands out

- Broad coverage from beginner markup to practical JavaScript logic.
- Clear one-question-per-file structure for fast review and grading.
- Includes complete mini-projects (college and tourism websites), not just isolated snippets.

## Assignment 1 snapshot (HTML + multipage websites)

- `q1` to `q7`: core HTML practice (structure, tables, text formatting, lists, media, forms).
- `q8-college-website/`: full college portal flow with pages for home, academics, departments, admission, facilities, rules, and contact.
- `q9-tourism-website/`: Hampi tourism mini-site with heritage pages, gallery, booking, and contact flow.
- `q10.html`: personal portfolio/resume-style page.
- `q11.html`: model question paper layout using semantic HTML elements.

## Assignment 2 snapshot (JavaScript problem set)

- `q1` to `q9`: searching, sorting, string ordering, and sorted-check logic.
- `q10` to `q14`: type checks, object mutation, and object-array iteration.
- `q15` to `q19`: timers, URL parsing, string transforms, and utility conversions.
- `q20` to `q22`: basic class-based data structures and geometry methods.
- `q23` to `q25`: image navigation logic, JSON error handling, and browser redirect behavior.

## Quick run guide

### HTML files

Open any file directly in a browser, for example:

```bash
xdg-open assignment-1/q10.html
```

### JavaScript files

Run with Node.js, for example:

```bash
node assignment-2/q3.js
```

## Evaluation-friendly design

- Predictable naming (`q<number>`) keeps navigation effortless.
- Solutions are intentionally direct and readable.
- Great for viva prep, assignment review, and quick concept recap.
